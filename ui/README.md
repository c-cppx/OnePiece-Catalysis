# onepiece-studio

`onepiece-studio` is the Streamlit frontend for the `onepiece` backend.

Install it when you want the local browser UI:

```bash
pip install onepiece-studio
```

This package depends on `onepiece` and adds the interactive command:

```bash
onepiece-studio
```

Useful entry points:

```bash
onepiece-studio tutorial
onepiece-studio demo
onepiece-studio qa
onepiece-studio doctor
```

Recommended first run for a new student:

```bash
onepiece-studio doctor
onepiece-studio qa
onepiece-studio tutorial
```
